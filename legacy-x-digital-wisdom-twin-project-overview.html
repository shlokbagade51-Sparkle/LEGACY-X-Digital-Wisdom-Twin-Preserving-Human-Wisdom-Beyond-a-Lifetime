Multi-Agent Architecture

👤

Legacy Agent

Speaks with the voice and wisdom of the preserved individual. Uses only uploaded memories. First-person, emotionally intelligent.

📊

Analyst Agent

Logical evaluation — Benefits, Risks, Alternatives, Reasoning. Evidence-based, structured, objective analysis.

⭐

Consensus Agent

Synthesizes both agents into one balanced, actionable recommendation. Highlights agreements and disagreements.

Quick Start

1

Install dependencies: pip install -r requirements.txt

2

Copy environment file: cp .env.example .env — fill in your IBM credentials

3

Run the app: python app.py → open http://localhost:5000

4

Go to Knowledge page, upload PDF/TXT/DOCX documents

5

Open Dashboard, ask questions — all 3 agents respond with grounded wisdom
